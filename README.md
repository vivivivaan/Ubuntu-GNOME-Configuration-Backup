# Ubuntu-24.04.1-LTS Gnome (46) Extensions/Config-BackUp.

Extensions are stored in ```~/.local/share/gnome-shell/extensions```.

# 

First install the **Extension Manager** from **Flathub** using `flatpak install flathub com.mattjakeman.ExtensionManager`. As this repo also contains the extensions with their configurations, just clone and copy all the contents into `~/.local/share/gnome-shell/extensions/`. To apply the configurations, navigate to **Aaaa Extensions Configurations Backup** folder and run `dconf load /org/gnome/shell/extensions/ < gnome-shell-extensions-backup.dconf`.
