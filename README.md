# Ubuntu-24.04.1-LTS Gnome 46 Extensions-BackUp.

Extensions are stored in ```~/.local/share/gnome-shell/extensions```.

# To install

First install the **Extension Manager** from **Flathub** using `flatpak install flathub com.mattjakeman.ExtensionManager`.
